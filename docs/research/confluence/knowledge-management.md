# Confluence: Knowledge Management Capabilities

## Overview
Confluence is Atlassian's AI-powered workspace for knowledge and collaboration. It serves as a centralized platform where teams create, organize, and share documentation, project plans, meeting notes, and knowledge base articles. Recognized as a Leader in the 2025 Gartner Magic Quadrant for Collaborative Work Management.

## Spaces
- Work is organized into "spaces" — each scoped to a project, department, team, or initiative
- Each space contains pages, databases, live docs, and blog posts
- Spaces mirror business functions, products, or teams for compartmentalized information
- Personal spaces available for individual use
- Best practice: separate spaces per product/service rather than one monolithic space
- Consistent structure across spaces allows users to navigate unfamiliar spaces intuitively
- Space-level permissions control who can view, edit, or administer content

## Pages & Content Hierarchy
- Hierarchical page structure with parent/child relationships
- Child pages nest below parent pages, creating categories and subcategories
- Any page can have its own child pages to arbitrary depth
- Dynamic pages support real-time co-editing
- Page versioning tracks all changes with full history
- Page status labels (e.g., Draft, Current, Archived) communicate content freshness
- Each page can have comments (inline and page-level) for discussion
- Pages support rich content: text, tables, images, videos, code blocks, diagrams, embeds

## Templates
- AI-powered page creation and ready-to-use templates
- Draft PRDs, social briefs, annual plans, and more in seconds
- Templates auto-apply labels to new pages created from them
- Custom templates can be created per space or globally
- Rule templates support low-code/no-code custom automation rules
- Template categories help users find the right starting point
- Community template library extends built-in options

## Labels
- Labels are keywords/tags applied to pages and attachments
- Organize and group related content across spaces
- Can add unlimited labels per page
- Labels applied while editing or after publishing
- Template labels automatically inherited by pages created from that template
- Click any label to discover all pages sharing that label
- Search by label name to filter content organization-wide
- Knowledge base space type comes pre-configured with Livesearch and Content By Label macros
- Labels serve as the primary lightweight taxonomy mechanism

## Macros
- Over 100 built-in macros extend page functionality
- Content By Label macro: dynamically lists pages matching specific labels
- Livesearch macro: embeds a search box scoped to a space
- Jira macro: embeds Jira issues/filters directly in pages
- Table of Contents macro: auto-generates navigation for long pages
- Expand macro: collapsible sections for information-dense pages
- Include Page macro: transcludes content from other pages
- Roadmap macro: visual timeline planning
- Chart macros: data visualization
- Third-party macros available via Atlassian Marketplace (thousands of apps)

## Permissions
- Granular permission model at space and page levels
- Space permissions: view, edit, create, delete, administer
- Page-level restrictions can override space permissions
- Group-based permissions for team management
- External sharing via restricted public links (no license required for viewers)
- Anonymous access configurable per space
- Admin controls for bulk permission management

## Integrations
- **Jira**: Deep bidirectional integration — link pages to issues, sprints, repositories
- **Trello**: Kanban-style task management embedded in Confluence
- **Slack**: Notifications, page previews, create pages from messages
- **Google Drive**: Embed and link Google Docs/Sheets/Slides
- **Loom**: Video communication embedded in pages
- **Rovo AI**: Cross-product AI-powered insights and search
- **Marketplace**: Thousands of plugins for forms, diagrams, workflows, and more
- Connect to third-party tools to access data and manage workflows across apps

## AI Features (Atlassian Intelligence / Rovo)
- AI-powered page and comment summaries
- Audio briefings for catching up on changes
- Content generation and editing assistance
- AI-powered search with instant answers and auto-suggestions
- Info tips that auto-define company-specific jargon and acronyms
- Natural language automation rule creation
- AI suggestions for adding labels on publish
- Cross-product AI via Rovo agent

## Collaboration Features
- Real-time co-editing on pages
- Inline and page-level comments
- @mentions to notify team members
- Watch pages/spaces for change notifications
- Activity feed for recent changes
- Blog posts for announcements within spaces
- Whiteboards for visual brainstorming

## Content Types
- Standard pages (hierarchical documentation)
- Blog posts (chronological announcements)
- Databases (structured data, introduced in recent updates)
- Live docs (lightweight, real-time collaborative editing)
- Whiteboards (visual collaboration)
- Smart links (rich previews of linked content)
